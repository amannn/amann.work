# amann.me

[![Build Status](https://travis-ci.com/amannn/amann.me.svg?branch=master)](https://travis-ci.com/amannn/amann.me)

My personal website featuring some projects I've worked on.

## Setup

1. Run `yarn install`
2. Add `GH_TOKEN` to `.env`

## Development

```sh
yarn dev
```

## Run in production

```sh
yarn build
yarn start
```
