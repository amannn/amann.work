# amann.work

[![Build Status](https://travis-ci.com/amannn/amann.work.svg)](https://travis-ci.com/amannn/amann.work)

My personal website featuring some projects I've worked on.

## Setup

1. Run `yarn install`
2. Add `GH_TOKEN` to `.env`

## Development

```sh
yarn dev
```

## Run in production

```sh
yarn build
yarn start
```
